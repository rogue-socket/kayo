# Rogue-Socket GitHub Profile — Yash Agrawal's Portfolio

## Summary
Yash Agrawal's GitHub profile README showcasing a portfolio focused on AI agents, structured reasoning, LLM backends, and evaluation pipelines.

The profile presents 11 projects spanning agent frameworks (closedclaw, ForrestRun), browser/phone automation agents (mycompagent, scrollable), domain-specific agentic flows (doctor_reporting, AquaTwin), content generation (agentic_article_writer), learning tools (math_guitar_VLM, learnwithme), a personal notes app (notes_me), and a from-scratch neural network (neural_network_numpy). The unifying thread is building AI systems that are "effective, efficient, and deliberate" — with a strong emphasis on production-readiness and evaluation ("vibes don't match all the time").

## Key Ideas
- Core focus areas: AI agents, structured reasoning, production-aware LLM backends, evaluation pipelines
- closedclaw is the most ambitious project — a full framework for agent creation, deployment, orchestration, and storage ("a literal village of agents which grows")
- ForrestRun is an agentic runtime for designing, fixing, and teaching agents
- mycompagent (browser) and scrollable (Android phone) are device-control agents using Playwright and similar tooling
- AquaTwin bridges AI agents with industrial digital twins — applicable to any digital twin interface
- doctor_reporting applies agentic flows to healthcare consultation report generation
- neural_network_numpy demonstrates deep fundamentals — building a generic NN with only NumPy
- Tech stack leans heavily on LangChain, LangGraph, tool-calling, knowledge graphs, and orchestration

## Insights & Claims
- Evaluation is a first-class concern, not an afterthought — the bio explicitly calls it out
- The portfolio shows a pattern of building agent "runtimes" and "frameworks" rather than one-off agents — systems thinking
- The range from browser automation to digital twins to healthcare shows cross-domain agent application
- "A literal village of agents which grows" suggests a multi-agent architecture philosophy where agents can be composed and scaled
